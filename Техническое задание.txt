Техническое задание на разработку программного обеспечения для прогнозирования хэштегов по фотографиям в профиле социальной сети
__________________________________________________________________________________________________________________________________

-Назначение разработки-
Разработать ПО, позволяющее пользователю загружать фотографии и получать релевантные хэштеги для публикации в социальной сети

-Платформа-
Desktop app/Telegram bot

-Этапы разработки-
1. Определить датасет для обучения
	- использовать готовый датасет (Harrison)
	- разработать программу и собрать с ее помощью данные
	- использовать https://instaloader.github.io/ для сбора данных
	Требуется датасет не менее 50000 фото
2. Исследовать и опробовать модели, произвести оценку и выбрать итоговую
	- Accuracy для топ-5 хэштегов не менее 52%, судя по бейзлайну датасета Harrison.
3. Разработать приложение для пользователя

-Логика работы итоговой системы-
При загрузке пользователем фотографии, система возвращает 10 релевантных хештегов.
